## Tasks

##  Concepts Covered

* `COUNT()` — Count total rows
* `SUM()` — Calculate total sum
* `AVG()` — Find average value
* `MIN()` — Get minimum value
* `MAX()` — Get maximum value
* `GROUP BY` — Group results by column
* `HAVING` — Filter grouped results
